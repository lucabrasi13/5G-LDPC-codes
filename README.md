# 5G-LDPC-codes
My course work for 5G LDPC codes

1. Dependency
> it++ for vector class. 
> GNUPLOT for plotting. 

2. Installation
> To install GNUPLOT
```
sudo pacman -S gnuplot
```
> To install itpp 
```
yaourt itpp
```
3. To run the code
> Compilation
```
g++ -o exec foo.cc -litpp
```
> Execution 
```
./exec
```
